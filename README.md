# 🧠 คลังพรอมต์ (Prompt Library)

คลังรวมพรอมต์คุณภาพสูงสำหรับสร้างภาพแนวแฟนตาซีด้วยเครื่องมือ AI เช่น DALL·E, ChatGPT, Midjourney และอื่น ๆ

## 📁 โครงสร้างโฟลเดอร์
/prompts
├── characters/       # พรอมต์ออกแบบตัวละคร
├── summons/          # พรอมต์สิ่งมีชีวิต คู่หู หรือสัตว์เวท
├── environments/     # พรอมต์ฉาก สถานที่ และแอ็กชัน
├── combo/            # พรอมต์ที่มีหลายตัวละครหรือฉากต่อสู้

## ✨ รูปแบบของแต่ละพรอมต์

ไฟล์แต่ละอันเป็น `.md` และประกอบด้วย:
- ชื่อเรื่อง
- หมวดหมู่และแท็ก
- กรณีการใช้งาน เช่น DALL·E, Midjourney
- ข้อความพรอมต์ในรูปแบบโค้ดบล็อก เพื่อให้นำไปใช้ได้ง่าย

ตัวอย่าง:

```text
Anime-style fantasy queen with long flowing hair, wearing an elegant white and silver gown. She wears a golden crown with icy gems, and has large black feathered wings fully spread. Sitting on a massive throne of clear ice...

✅ ใช้คลังนี้เพื่อ:
	•	เก็บพรอมต์ที่คุณชื่นชอบ
	•	เรียกใช้งานซ้ำหรือนำไปดัดแปลง
	•	แชร์ให้เพื่อนหรือทีมใช้งานร่วมกัน
	•	จัดการ workflow ด้านครีเอทีฟอย่างเป็นระบบ

💬 การใช้งานเพิ่มเติม

นี่คือคลังส่วนตัวหรือของทีม แต่คุณสามารถ fork ไปสร้างคลังของคุณเอง
